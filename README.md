# murmuration-writing-companion

**A focused writing companion for Obsidian.**

Writing a novel is difficult enough. Your tools shouldn't make it harder.

Murmuration Writing Companion is an open-source plugin for Obsidian that helps authors organise the editorial process without interrupting the act of writing.

It doesn't try to replace your writing environment.

It doesn't lock your work into a proprietary format.

It doesn't tell you how to write.

Instead, it quietly supports the author with document notes, editorial workflow, checklists and contextual information, while leaving the manuscript as clean, portable Markdown.

---

## Philosophy

The companion is built around a few simple ideas.

* The manuscript is sacred.
* Authors own their work.
* Plain text first.
* Open formats over proprietary formats.
* Reduce cognitive load.
* Small, focused applications are better than monolithic software.
* Software should disappear while writing.

If a feature doesn't support those principles, it probably doesn't belong here.

---

## Current Features

* Document-level editorial notes
* Selection-anchored notes
* Scrivener-style inspector panel
* Editorial categories
* Resolve/reopen workflow
* Portable Markdown-first workflow
* Linux-first development with cross-platform support

## The Murmuration Ecosystem

Murmuration Writing Companion is part of a wider ecosystem of focused tools for authors.

* **Murmuration Press** — Independent publisher.
* **Murmuration Writing Companion** — Writing support inside Obsidian.
* **Codex Press** — Publishing tools for producing EPUB and PDF from Markdown.

Each project has a single responsibility.

Together, they provide a complete writing and publishing workflow built entirely on open formats.
