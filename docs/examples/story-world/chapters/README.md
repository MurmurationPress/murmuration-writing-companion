# Chapter World Context examples

These examples demonstrate the `world_context` convention defined by the [Chapter World Context Standard](../../../chapter-world-context-standard.md).

- [`simple-world-context.md`](simple-world-context.md) combines an automatically recognised POV entity with one explicit context entity.
- [`multi-entity-world-context.md`](multi-entity-world-context.md) references a character, organisations, an event and an intelligence while preserving the existing free-text location.

They are schema illustrations only. They are not copied into the PRIME Trilogy vault and do not independently establish canon.
