# Supporting model examples

These examples exercise the conventions in [`docs/supporting-model-conventions.md`](../../../supporting-model-conventions.md).

- `tobias-northbridge-employment.md` — a time-bounded relationship with an unsettled end.
- `routing-before-halcyon.md` — an author-maintained relative timeline assertion.

The simple entity-owned relationship is stored on [`../pip.md`](../pip.md), with [`../robin.md`](../robin.md) demonstrating a derived inverse that is not duplicated in Markdown.

Every example includes a plain-language rendering. The examples are documentation only and do not independently establish PRIME Trilogy canon.
